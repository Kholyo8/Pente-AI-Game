# 🎮 Pente AI Game

A Python-based implementation of the classic Pente board game with an AI opponent.

---

## 🚀 Features
- Interactive graphical interface for playing Pente  
- AI opponent with decision-making logic  
- Modular code structure (Board, AI, UI)  
- Full implementation of game rules (captures & win conditions)  

---

## 🧠 AI Logic
The AI uses heuristic-based decision making to:
- Evaluate board positions  
- Block opponent moves  
- Attempt winning strategies  

---

## 🛠️ Tech Stack
- Python  
- Object-Oriented Programming (OOP)
